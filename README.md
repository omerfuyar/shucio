# Shucio
Shucio (SHU Console Input Output) is a cross platform single header library to get instant user input (like ncurses) from any console in Windows, Linux and MacOS with additional features. Plus, I am not planning to add any windows or lists or anything you get from ncurses except the input and some output features.

# Usage & Examples
I have a small terminal game project that uses shucio, you can check it out [here](https://github.com/omerfuyar/NewCharlie). Soon I am planning to add more examples like I did in [shuild](https://github.com/omerfuyar/shuild).
